# AgriConnect
# 🌾 Farm E-Commerce Website

A web-based **E-Commerce Platform for Farmers** developed using **Flask (Python)**.  
This application allows farmers to sell agricultural products directly to customers, while admins manage users, products, and orders.

---

## 📌 Project Overview

The Farm E-Commerce Website provides a simple and user-friendly platform where:
- Farmers can log in and add products
- Customers can browse products and place orders
- Admin can manage users, products, and orders

The system reduces the role of middlemen and supports direct farmer-to-consumer sales.

---

## 🚀 Features

### 👨‍🌾 Farmer Module
- Farmer registration and login
- Add new products
- Manage inventory
- View sales details
- Logout

### 🛒 Customer Module
- User registration and login
- Browse agricultural products
- Add products to cart
- Checkout and payment
- View order confirmation

### 🛠 Admin Module
- Admin login
- View and manage users
- Manage products
- View customer orders
- Logout
